# Tic Tac Toe Game

A simple terminal-based Tic Tac Toe game implemented in C++. The game features an AI opponent that uses the minimax algorithm to play optimally, ensuring it never loses. Depending on the moves, the AI can either win or draw.

## Features
- Single-player mode: Human vs Computer AI
- Player can choose their symbol (X or O)
- Player can choose to go first or let the computer go first
- AI implemented with minimax algorithm and a custom fallback algorithm
- Clear terminal interface for gameplay

## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/YOUR_USERNAME/tic-tac-toe.git
    ```

2. Compile the C++ files using a compiler like g++:

    ```bash
    g++ *.cpp -o tic_tac_toe
    ```

3. Run the executable:

    ```bash
    ./tic_tac_toe
    ```

## Usage
- Follow on-screen prompts to play.
- Choose your symbol and whether you go first.
- Input moves via keyboard as prompted.

## Technologies Used
- C++ (with standard libraries)
- Terminal/console input-output
